# Project Archive

Archived scripts from past sequencing projects, organized by assay type.

## Structure

| Folder | Assay | Contents |
|--------|-------|----------|
| **ATACseq/** | ATAC-seq | snATAC HepG2 (ENCODE Snyder) |
| **ChIPseq/** | ChIP-seq | ENCODE/GEO Bernstein, Myers, Snyder; lab projects (Likun, Yuliang) |
| **CUTRUN/** | CUT&RUN | Lab CUT&RUN projects (Steven, etc.) |
| **PROseq/** | PRO-seq | PRO-seq and scRNA projects |

## Usage

- **Reference only**: these are one-off, project-specific scripts.
- **Templates**: for new projects, use `pipeline/cutrun/` (standard pipeline) instead.
- Folder names follow: `YYMMDD_Assay_CellLine_Source` or `Assay_Description_Source`.
